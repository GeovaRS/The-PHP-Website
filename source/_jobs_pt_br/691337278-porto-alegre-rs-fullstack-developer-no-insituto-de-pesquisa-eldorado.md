# [Porto Alegre -RS] FullStack Developer no Insituto de Pesquisa Eldorado

> Vaga Remota durante a pandemia (previsão de retorno em janeiro)

## Nossa empresa
 O Eldorado é atuante na área de P&D de Software e Hardware e classificado como uma das Melhores Empresas para Trabalhar, com projetos globais.

## Descrição da vaga

**Atividades:**
Atuação no desenvolvimento de solução de aplicação global, em conjunto com parceiro multinacional, com foco técnico no desenvolvimento de soluções backend/frontend Web.

**Local:**
Porto Alegre - RS

**Requisitos:**
- Formação superior em Engenharia/Ciência da Computação, Sistemas de Informação ou similar;
- Experiência com desenvolvimento de software para soluções Web;
- Experiência com arquitetura de aplicações Web;
- Experiência com desenvolvimento em JAVA e framework Spring Boot;
- Experiência com desenvolvimento em Javascript e frameworks CSS;
- Experiência com banco de dados relacionais;
- Experiência com Hibernate;
- Inglês para conversação, escrita e leitura.

**Desejáveis:**
- Experiência com metodologias ágeis (Agile Scrum);
- Conhecimento com GIT, Gerrit e Jenkins tools;
- Conhecimento com RESTful Web Services;
- Conhecimento com AngularJS;
- Conhecimento com Google Cloud Platform (App Engine, Kubernetes, Cloud SQL, Data Store, Big Query);
- Conhecimento com banco de dados não relacionais;
